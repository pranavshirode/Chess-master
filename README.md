# Chess (Chess-master)

Compact, maintainable chess project providing game rules, move generation, and optional UI/CLI. This repository contains the game engine, AI hooks, tests, and tooling.

Key contents
- src/ — core engine and modules
- tests/ — unit and integration tests
- docs/ — extended documentation

Quick Start (dev)
1. Install prerequisites: Node.js (or relevant runtime), and any build tools (see docs/INSTALLATION.md).
2. Install deps: run the package manager for this project (see docs/INSTALLATION.md).
3. Run tests: see docs/TESTING.md.
4. Run the app: see docs/USAGE.md.

Where to look first
- docs/ARCHITECTURE.md — architecture overview and module map
- docs/USAGE.md — how to run and play
- docs/CONTRIBUTING.md — contribution workflow
